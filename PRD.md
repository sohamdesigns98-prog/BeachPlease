# BeachPlease PRD

## Overview

BeachPlease helps users discover and plan beach days based on what they are craving.

## Initial Scope

- Minimal React + Vite frontend.
- FastAPI backend with a health endpoint.
- CORS configured for local frontend development.

## Out of Scope for Initial Skeleton

- Routing
- API calls from the frontend
- Map UI
- Ticket UI
- MongoDB
- Authentication
- Gemini
- Open-Meteo
- CRUD flows
